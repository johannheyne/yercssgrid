YerCSSGrid

The main structur of classe are {classtype}-{breackpoint}-{value}.

Conventions

Because of the technic to layout the gaps, we need overflow hidden on the body. And because of all is fluid and relative, we need border-box as box-sizing on all elements.

body { overflow: hidden; }
* { box-sizing: border-box; }

Using Columns

Columns with gap

In the following example we have a gap of 20 pixel starting on browser-width of 0. The cols starting at browser-width of 500 pixel.

<div class="grid gap-0-20">
  <div class="col-500-6"></div>
  <div class="col-500-6"></div>
</div>

Columns with gap
